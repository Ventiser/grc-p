# RGIS v1.0
Registry–Gatekeeper Interface System

Status: Draft  
Parent Standard: GRC-P  
Maintainer: Ventiser  

## Abstract

RGIS defines the runtime interoperability protocol between registry authorities and enforcement gatekeepers under the GRC-P governance architecture.

The protocol determines whether operational actions are admissible before execution occurs.

Gatekeepers consult registry authority state and return deterministic permit/deny decisions.
